# Moura & Queiroz Advocacia

Segundo projeto demonstrativo de advocacia publicado no mesmo repositório e no mesmo processo de GitHub Pages, sem alterar o site Almeida & Castro.

## Site publicado

`https://ricardoribeiro-prof.github.io/almeida-castro-advocacia/moura-queiroz-advocacia/`

## Identidade

- Nome: Moura & Queiroz Advocacia
- Paleta: verde-floresta, terracota, creme e branco
- Estilo: editorial, contemporâneo, sóbrio e acolhedor
- Localização demonstrativa: São Raimundo Nonato – PI
- Logotipo: monograma monocromático M&Q
- Modo: demonstração, com `noindex, nofollow`

## Estrutura

O projeto possui 16 páginas públicas pré-renderizadas e uma página 404:

- página inicial;
- escritório;
- áreas de atuação;
- seis páginas jurídicas;
- equipe;
- artigos;
- três artigos completos;
- contato com direcionamento para WhatsApp;
- política de privacidade;
- página 404.

## Arquivos principais

- `scripts/moura-queiroz-content.mjs`
- `scripts/generate-moura-queiroz-site.mjs`
- `scripts/audit-moura-queiroz.mjs`
- `public/moura-queiroz-advocacia/assets/styles.css`
- `public/moura-queiroz-advocacia/assets/app.js`
- `public/moura-queiroz-advocacia/assets/favicon.svg`
- `public/moura-queiroz-advocacia/assets/social-share.svg`

O comando `npm run build` gera o projeto dentro de:

`dist/moura-queiroz-advocacia/`

A auditoria local e publicada é executada automaticamente pelo workflow permanente do GitHub Pages.
